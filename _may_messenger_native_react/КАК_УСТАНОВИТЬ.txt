📋 ШАГ 2: РАЗРЕШИТЕ УСТАНОВКУ

На телефоне:
1. Откройте Настройки → Безопасность
2. Найдите "Неизвестные источники" или "Установка из неизвестных источников"
3. Разрешите установку для вашего файлового менеджера/браузера

Для Samsung (как на вашем телефоне):
Настройки → Биометрия и безопасность → Установка неизвестных приложений
→ Выберите приложение (Мои файлы/Chrome) → Разрешить

═══════════════════════════════════════════════════════

📋 ШАГ 3: УСТАНОВИТЕ ПРИЛОЖЕНИЕ

1. Откройте файл depesha-v1.0-release.apk на телефоне
2. Нажмите "Установить"
3. Дождитесь завершения установки
4. Нажмите "Открыть" или найдите иконку "Депеша" в списке приложений
